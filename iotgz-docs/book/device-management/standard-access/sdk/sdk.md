# **设备接入SDK**

物联平台提供设备接入SDK适用于“MCU+标准通信模组”、单板SOC、java平台的方案。SDK由固定代码包和平台生成的配置文件构成，用户根据使用的硬件平台集成SDK，通过配置选择不同的接入协议，并调用相应API接口即可实现设备的快速接入。
