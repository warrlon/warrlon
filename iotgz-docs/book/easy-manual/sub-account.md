# 子账号概述

为满足员工或企业可控制同一主账号下不同的产品资源权限。OneNET平台提供了子账号功能，您可以集中管理您的子账号信息（比如员工、企业等），也可管理子账号访问主账号名下的产品资源权限。

每个主账号最多可创建12个子账号，子账号忘记密码后可通知主账户管理员重置密码。
