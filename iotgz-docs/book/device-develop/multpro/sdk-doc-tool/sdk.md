#### LwM2M的SDK获取

```

公司（厂商）名称：中移物联网公司
公司地址：重庆市渝北区杨柳路xxx号
即将移植的芯片或模组型号：M311
联系人名称及职务:张三
联系电话：180xxxxxxxx
邮箱：zhangshan@xxx.com
```
