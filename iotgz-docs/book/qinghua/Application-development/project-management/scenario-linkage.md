# 场景联动

用户点击【进入项目管理】按钮，在跳转页面点击【场景联动】导航栏，进入场景联动页面，主要包括规则名称、规则描述、启用状态创建时间信息。

![场景联动](/images\qinghua\device-management\scenario-linkage.png)

该页面支持以下功能：

- 支持禁用/启用规则功能
- 支持编辑规则功能
- 支持日志查询功能
- 支持删除场景功能
- 支持添加规则功能

其中，点击【编辑规则】按钮可跳转至【规则内容】详情页，可查看已有规则内容。用户点击【编辑规则】按钮，可通过执行限制、触发方式、数值判断条件、选择动作类型、功能等内容设置触发条件和执行动作。

<div align=center><img src ="/images\qinghua\device-management\sl-rule-edit.png"/></div>

用户切换点击【联动日志查询】按钮，可通过时间范围进行查询，点击某一具体联动规则中【日志详情】按钮，即可查看日志详情。

![联动日志查询](/images\qinghua\device-management\sl-log-query.png)
