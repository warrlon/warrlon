# OneNET LBS位置能力

OneNET LBS位置能力为用户提供高效、准确的定位服务，目前支持基站定位服务与WiFi定位服务，方便快捷的接入使用流程，超大的免费日配额，低成本的使用方式，让物联网企业可以更加灵活的对智能硬件进行位置管理，从而降低企业研发、运营和运维成本。
