# 查看下载设备文件接口

#### 接口名称

> GetDeviceFile

#### 请求方法

> Get

#### 请求参数

| 参数名称   | 类型   | 是否必选 | 描述   |
| :--------- | :----- | :------- | :----- |
| id | string | 是       | 文件ID |

#### 返回数据

| 参数名称          | 类型    | 描述                       |
| :---------------- | :------ | :------------------------- |
| ---              | ---  | ---   |

#### 示例

##### 请求示例

```
GET /application?action=GetDeviceFile&version=1&id=43bb54ac673f48c88300fa6e6d3c9481
```


##### 响应示例

```
file
```
