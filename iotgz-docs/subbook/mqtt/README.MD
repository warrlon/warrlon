# MQTT物联网套件

MQTT物联网套件为物联网开发者提供高效、稳定、安全的平台服务，具有设备接入、设备管理、数据存储、状态同步、消息分发能功能，使物联网企业可以更加专注于自身应用的开发，而不用将工作重心放在设备接入层的环境搭建上，从而缩短物联网系统的形成周期，降低企业研发、运营和运维成本
